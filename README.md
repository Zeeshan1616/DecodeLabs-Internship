## 📌 Overview

This is **Project 1** of my Python Programming internship at **Decode Labs** (2026 batch).

The app lets a user add tasks to a list and view them through a simple menu, all running in the console. The core idea behind it is learning how to store multiple items in a single variable using Python lists, a foundational concept before moving on to databases.

---

## ✨ Features

- ➕ **Add Task** — add a new task to your list
- 📋 **View Tasks** — see all tasks, neatly numbered
- 🚪 **Exit** — close the program safely

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed on your machine

### Run the app
```bash
python todo_list.py
```

Then follow the on-screen menu to add or view your tasks.

---

## 🧠 Concepts Used

| Concept | Where it's used |
|---|---|
| Lists | Storing all tasks in `tasks = []` |
| `append()` | Adding new tasks to the list |
| Loops + `enumerate()` | Displaying tasks with numbering |
| Functions | Organizing logic into `add_task()`, `view_tasks()`, `main()` |
| `if __name__ == "__main__":` | Proper program entry point |
